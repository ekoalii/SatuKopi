Sistem CRUD pada aplikasi penjualan menggunakan framework laravel

1. Klik Login
2. Klik admin dan lakukan login masukkan username dan password admin.
3. Akan masuk ke tampilan admin.
4. Pada data barang admin dapat menambah, mengupdate, melihat, dan menghapus suatu data.
5. Pada tampilan invoice dapat melihat data - data yang sudah tertera suatu orderan.
6. Logout.
